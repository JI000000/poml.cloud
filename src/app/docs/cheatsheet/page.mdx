# POML Cheatsheet

## Core tags

- `<role>`: Define the assistant persona.
- `<task>`: Describe the goal or instruction.
- `<example>`: Provide few-shot examples.
- `<document>` / `<table>` / `<img>`: Reference external inputs.
- `<stylesheet>`: Control presentation separately from content.
- `<let>` / `if` / `for`: Templating helpers.

## Minimal example

```xml
<poml>
  <role>You are a helpful assistant.</role>
  <task>Summarize the document into 3 bullets.</task>
  <document src="summary.txt" />
  <output-format>- Bullet 1\n- Bullet 2\n- Bullet 3</output-format>
</poml>
```
