# Data Embedding in POML

## Components
- `<document src>`: text files, snippets and large context.
- `<table src>`: CSV or simple TSV.
- `<img src>`: images referenced by URL.

## Tips
- Prefer stable URLs; fall back to uploads if available.
- Normalize tables; declare units and headers.
- Pair with `<stylesheet>` to control output format.

Official docs: https://microsoft.github.io/poml/latest/language/standalone/
